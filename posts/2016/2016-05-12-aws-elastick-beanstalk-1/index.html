AWS Elastick Beanstalk 1

awsebcli 설치

linux

$ sudo apt install python3-pip
$ pip3 install awsebcli

MacOS

$ brew install python3
$ sudo pip3 install awsebcli

pip가 업데이드안되어서 설치가 안될경우에는

$ sudo pip3 install --upgrade pip

설치 시 sudo 를 사용하지 않고 로컬(자신의 홈디렉토리)에 설치 할경우에는
–user 파라미터를 사용하면 .local/bin 에 설치된다.
그외에 추가로 setuptools를 환경에 따라 요구할 수 있다.

eb 초기화

$ mkdir eb
$ cd eb
$ eb init

초기화를 실행하면 자신의 홈디렉토리에 .aws 라는 디렉토리가 생기므로 지워선 안된다.

Environment Configuration

Nginx

nginx.conf

eb에서 기본으로 설정된 파일을 복사한것임

# Elastic Beanstalk Nginx Configuration File
 
user                    nginx;
error_log               /var/log/nginx/error.log warn;
pid                     /var/run/nginx.pid;
worker_processes        auto;
 
events {
    worker_connections  1024;
}
 
http {
    include       /etc/nginx/mime.types;
    default_type  application/octet-stream;
 
    log_format  main  '$remote_addr - $remote_user [$time_local] "$request" '
                      '$status $body_bytes_sent "$http_referer" '
                      '"$http_user_agent" "$http_x_forwarded_for"';
 
    map $http_upgrade $connection_upgrade {
        default     "upgrade";
    }
 
    include       conf.d/*.conf;
 
}

web.conf

upstream backend {
    server 127.0.0.1:5000;
    keepalive 32;
}

server {
    listen       80;
    server_name  admin.oho.co.kr;
    access_log  /var/log/nginx/web.log  main;

    client_header_timeout 60;
    client_body_timeout   60;
    client_max_body_size  200M;
    keepalive_timeout     60;
    gzip                  on;
    gzip_comp_level       4;
    root                  /var/app/current/public;
    
    location ~* /(images|css|fonts)/*$ {
        expires 7d;
        access_log off;
        break;
    }

    location ~* \.(html?|css|js)$ {
        expires -1;
        access_log off;
        add_header Cache-Control no-cache;
        break;
    }

    location / {
        try_files           $uri @backend;
    }
                 
    location @backend {        
        proxy_set_header    Connection          $connection_upgrade;
        proxy_set_header    Upgrade             $http_upgrade;
        proxy_set_header    Host                $host;
        proxy_set_header    X-Real-IP           $remote_addr;
        proxy_set_header    X-Forwarded-For     $proxy_add_x_forwarded_for;
        proxy_pass          http://backend;
        proxy_http_version  1.1;
        proxy_redirect      off;
    }
}

Linux Server

.ebextensions/
├── 00_timezone.config
├── 01_yum.config
└── nginx
    ├── conf.d
    │   └── web.conf
    └── nginx.conf

00_timezone.config

commands:
 set_time_zone:
  command: ln -sf /usr/share/zoneinfo/Asia/Seoul /etc/localtime

01_yum.config

commands:
  yum-install:
    command: yum install -y ld-linux.so.2

eb deploy

Java로 개발하고 Fat Jar를 생성했다
배포하기 위해 Procfile를 작성한다

Procfile

web: java -Dserver.home=/var/app/current -Djava.library.path=/var/app/current/bin -jar start.jar

EB 사용가능한 환경보기

$ eb list

배포할 어플리케이션 환경을 선택한다

$ eb use ecfront

배포명령어를 입력한다

$ eb deploy
