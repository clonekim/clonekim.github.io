Docker 시작하기

이미지다운로드

$ docker pull [이미지이름]

컨테이너 생성

$ docker run --name [컨테이너이름]

컨테이너로 접속

$ docker exec -it [컨테이너] bash

컨테이너 시작/재시작

$ docker start [컨테이너이름]
$ docker stop  [컨테이너이름]

MySQL 사용하기

mysql이미지를 다운로드와 함께 컨테이너 시작하기

$ docker run --name mysql -e MYSQL_ROOT_PASSWORD=root -e TZ=Asia/Seoul -p 3306:3306 -d mysql --character-set-server=utf8

컨테이너확인

$ docker ps
