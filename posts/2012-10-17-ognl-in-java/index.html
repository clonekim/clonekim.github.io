OGNL

Maven 

 <dependency>
    <groupId>ognl</groupId>
    <artifactId>ognl</artifactId>
    <version>3.0.5</version>
</dependency>

소스


Map<String, Object> map = new HashMap<String, Object>();
map.put("book", "모바일 웹앱");

System.out.println(Ognl.getValue(Ognl.parseExpression("book"), map));

Map context = new HashMap();
List<String> values1 = (List)Ognl.getValue("{123,'abc','00oo'}", context, new Object());

System.out.println(values1);

Map values2  = (Map) Ognl.getValue("#{123:333,'abc':111, 'bbb':'Kim', 'other':'[0,2,3,4,5]'}", context, new Object());

System.out.println(values2);

출력

모바일 웹앱
[123, abc, 00oo]
{123=333, abc=111, bbb=Kim, other=[0,2,3,4,5]}
