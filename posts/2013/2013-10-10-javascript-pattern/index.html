Module Pattern

로직과 private 변수를 캡슐화하는 방법으로 아래와 같은 모듈패턴( 익명함수) 을 많이 사용한다.

 (function() {
  /* ... */
 })();

jQuery와 연동해서 사용할 경우 아래와 같이 파라미터를 전달하도록 한다.

 (function($) {
  /* ... */
 })(jQuery);

글로벌 변수를 전달해서 네임스페이스로 확장 할 수 있다.

 (function($, exports) {
  exports.Foo = "wem";
 })(jQuery, window);

그러므로 아래와 같은 패턴을 즉, 즉시 실행함수로 작성한다.

var global = (function() {
 return this;
