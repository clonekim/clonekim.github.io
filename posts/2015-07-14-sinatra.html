설치

jruby -S gem install sinatra

Hello Ruby

require 'rubygems'
require 'sinatra'

get '/' do
  'Hello Jruby'
end

라우터

get, post, put, delete를 사용할 수있다

get '/' do

end

요청파라미터

/?name=foo 로 요청 시 아래와 같이

post '/' do
  puts params['name']
end

본문으로 Json를 요청받을 경우

post '/' do
  payload = JSON.parse(request.body.read)
  puts payload
  status 201
end

{{:jruby:post.png?200|}}

응답하기

문자열, html, json, 바이너리 등의 모든 컨텐츠로 응답가능하다

get '/' do
  'Hello'
end

Json으로 응답하기

require 'sinatra'
require 'json'

get '/' do
  content_type :json
  data = { foo: 'bar'}
  data.to_json
end

또는 sinatra-contrib를 사용해서

require 'sinatra'
require 'sinatra-contrib'

get '/' do
  data = { foo: 'bar'}
  json data
end

Sample Web

app.rb

require 'rubygems'
require 'sinatra'
require 'sinatra/reloader'
require 'json'
require 'sequel'
require './mysql-connector-java-5.1.35.jar'

DB = Sequel.connect('jdbc:mysql://127.0.0.1:3306/test?useUnicode=true&characterEncoding=utf8',
			:user=>'test',
			:password=>'test')
user = DB[:users]


get '/' do
  erb :index
end

get '/api/users' do
  data = user.all
  pager = {
 	total: data.count,
	rows: data,
	page: 1
  }
  pager.to_json
end

config.ru

require 'app.rb'
run Sinatra::Application

아래와 같이 실행한다

jruby app.rb

실행 시 내장된 웹서버서(WEBrick)를 사용한다
기본적으로 4567번 포트를 사용한다

아래는 Rack기반으로 기동하는 방법이다
기본적으로 9292번 포트를 사용한다

jruby -S rackup config.ru

===== 템플릿 =====

기본적으로 erb템플릿을 사용한다

디렉토리 구조

public에 정적인 파일을 views에 .erb파일이 위치한다.

├─public └─views

   index.erb
   layout.erb

<%= yield %>

Hello Sinatra

<%= $servlet_context %>

get '/' do erb :index end

$servlet_context라는 인스턴스명을 출력할 것이다 \ 단지 rack기반일 경우에만 출력됨

Contrib 활용

jruby -S gem install sinatra-contrib

reloader만 추가하고 소스를 수정 시 바로 적용됨

require 'sinatra/reloader'

War 배포

warbler를 설치한다

jruby -S gem install warbler

루비웹 프레임워크는 rack 이라는 인터페이스위에서 작동한다
자바의 서블릿과 같다
rack으로 기동하는 파일을 작성한다

require 'app.rb' run Sinatra::Application

config 디렉토리를 만들고 warble config를 작성한다 \ config아래에 warble.config라는 파일이 생성된다

mkdir config jruby -S warble config

Warbler::Config.new do |config|

config.dirs = %w(public views) config.includes = FileList["app.rb"] config.gems += ["sinatra","sinatra-contrib", "sequel"] config.gem_dependencies = true config.pathmaps.application = ["WEB-INF/%p"] config.jar_name = "app" end

이제 war파일을 작성하면 \ app.war 라는 파일이 보일것이다 jruby -S warble
