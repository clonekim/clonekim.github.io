Clojure를 위한 Emacs 환경

2015-09-11

Emacs 24를 기준으로 작성
현재 사용중인 환경은 custom-emacs.tgz

Package Install

init.el

(require 'package)
(add-to-list 'package-archives
        '("melpa-stable" . "http://stable.melpa.org/packages/") t)

(package-initialize)

package install

M-x package-install [RET] cider [RET]

Cider 구동

주의

Cider와 cider-nrepl의 버전이 맞아야함

profiles.clj

.lein디렉토리이하에 작성한다 없을경우 파일생성

{:user 
 {:plugins 
  [[cider/cider-nrepl "0.10.0"]]
  :dependencies
  [[org.clojure/tools.nrepl "0.2.12"]]}}
{:repl
 {:dependencies
  [[org.clojure/tools.nrepl "0.2.12"]]}} 
M-x cider-jack-in

단축키

C-c M-j: cider-jack-in
C-c C-k: compile
C-c M-n: 현재 작업중인 namespace로 변경
