윈도우즈 Vim설정

2015-07-16

vi를 윈도우즈에서 사용하기 위한 가이드

설치

  • http://www.vim.org
  • http://www.vim.org/download.php

설정

아래 파일을 C:\Users[계정명]_vimrc 에 위치한다

set nu
set autoindent
set nobackup
set showcmd
set ruler
set enc=utf-8
set fencs=euckr,cp949,utf-8
set tabstop=3
set nowrap
set lines=30 columns=140

lang mes en_US

source $VIMRUNTIME/delmenu.vim
source $VIMRUNTIME/menu.vim

Explore

명령모드에서 :Explore로 입력하면 탐색창모드로 전환한다. 특정파일을 선택 시 편집할 수 있고
Ctrl +^ 입력시 바로 이전 탐색창으로 이동한다.
