서버스크립트

VNC서버를 자동으로 시작하기 위해 첨부파일 스크립트를 사용한다
스크립트를 복사후 실행권한을 준다

tightvnc

### BEGIN INIT INFO
# Provides: tightvnc
# Required-Start: $remote_fs $syslog
# Required-Stop: $remote_fs $syslog
# Default-Start: 2 3 4 5
# Default-Stop: 0 1 6
# Short-Description: Start VNC Server at boot time
# Description: Start VNC Server at boot time.
### END INIT INFO

#! /bin/bash
# /etc/init.d/tightvnc

USER=pi
HOME=/home/pi

DISPNO=1       # Display No
WIDTH=1280     # Display geometry widht
HEIGHT=800     # Display geometry height
DEPTH=24       # Use color depth

OPTIONS=":${DISPNO} -geometry ${WIDTH}x${HEIGHT} -depth ${DEPTH}"

export USER HOME

case "$1" in
    start)
        echo "Starting TightVNC Server"
        #Insert your favoured settings for a TightVNC session
        su $USER -c "/usr/bin/tightvncserver ${OPTIONS}"
        ;;

    stop)
        echo "Stopping VNC Server"
        su $USER -c "/usr/bin/tightvncserver -kill :$DISPNO"
        ;;

    *)
        echo "Usage: /etc/init.d/tightvnc {start|stop}"
        exit 1
        ;;
esac

exit 0

sudo cp tightvnc /etc/init.d
sudo chmod +x /etc/init.d/tightvnc

1. 서버를 시작한다

sudo /etc/init.d/tightvnc start

2. 서버 부팅시 자동으로 실행될 수 있도록 작성

sudo update-rc.d tightvnc defaults
