Deployd

2017-08-21

몽고디비를 이용한 JSON Dummy 서버툴이다.

설치

$ npm install -g deployd

몽고디비

$ docker run --name deployd-mongo -d mongo -p 27017:27017

Deployd 실행

$ dpd create [project name]
$ cd [project name]
$ dpd -H localhost -P 27017 -d
