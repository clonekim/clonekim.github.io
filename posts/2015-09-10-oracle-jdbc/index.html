Oracle Jdbc Url

JDBC URL syntax

jdbc:oracle:thin:@[HOST][:PORT]:SID

jdbc:oracle:thin:@//[HOST][:PORT]/SERVICE

Hikari Pool를 사용 시 아래와 같이 경고메시지가 나타나면
oracle.jdbc.driver.OracleDriver에서 oracle.jdbc.OracleDriver 로 바꿀 것

2015-09-10 WARN  (main) [DriverDataSource(<init>:68)] Registered driver with driverClassName=oracle.jdbc.driver.OracleDriver was not found, trying direct instantiation.
2015-09-10 DEBUG (main) [PoolUtilities(getAndSetNetworkTimeout:198)] HikariPool-0 - Connection.setNetworkTimeout() not supported
