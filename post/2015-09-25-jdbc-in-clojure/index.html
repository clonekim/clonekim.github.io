클로저에서 Jdbc

클로저에서 JDBC(Java Database Connectivity) 를 사용하기 위한 가이드
몇몇 함수가 없어진 경우가 있기 때문에 소스보기로 확인할 것

  • clojure-doc보기
  • 소스보기

leiningen

[org.clojure/java.jdbc "0.4.2"]

간단한 예제

(ns db-test.core
	(:require [clojure.java.jdbc :as sql]))	

(def db
  {:classname "com.mysql.jdbc.Driver"
   :subprotocol "mysql"
   :subname "//127.0.0.1:3306/test"
   :user "xxx"
   :password "xxxx" })

(defn get-user []
  (println (sql/query db ["select * from users"]))) 

(defn -main [] 
  (get-user))
