java.lang.UnsupportedClassVersion_error
(Unsupported major.minor version 51.0)

version 52.0   컴파일 버전 : 1.8
version 51.0   컴파일 버전 : 1.7
version 50.0   컴파일 버전 : 1.6
version 49.0   컴파일 버전 : 1.5
version 48.0   컴파일 버전 : 1.4
