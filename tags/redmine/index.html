Redmine3 설치하기
2015-07-18
## 설치환경 redmine-3.0.4 centos 6.6 redmine install가이드 RVM 설치 rvm.io 에 대하여 rvm을 이용하여 루비를 버젼별로 설치및 관리할 수 있다 rvm install ruby rvm gemset create redmine rvm gemset use redmine 의존성 라이브러리 CentOS 의존성 redmine 설치 $ cd redmine-x.x.x $ gem install bundler $ bundle install --without development test Thin Gemfile을 수정하여 thin을 최상단에 추가 다시 번들로 라이브러리 설치 bundle install --without development test 아래 파일을 /etc/thin/redmine.
