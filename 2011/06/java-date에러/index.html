Java Date에러

Override time zone

생성한 Date 객체가 KST가 아닌 자꾸 GMT로 출력 될 경우

System.setProperty("user.timezone", "Asia/Seoul")

ex)
System.setProperty("user.timezone", "UTC");
TimeZone.setDefault(null); //reset 하기 위해 반드시 필요함

서버를 구동시에 -D 옵션으로 설정할 것
